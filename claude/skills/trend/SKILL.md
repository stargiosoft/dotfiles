# 트렌드 서칭 스킬 (Trend Search Skill)

> 멀티 플랫폼에서 실시간 트렌드를 크롤링하고 분석합니다.
> 캐릿 2026 트렌드 흐름 지도 기반 — 분야별 최적 플랫폼에서 서칭합니다.

---

## 사용법

```
/trend                         → 한국 실시간 전체 트렌드 (X + SNS 6개 + 웹)
/trend 운세 타로                → 특정 주제 서칭
/trend F&B 글로벌               → 글로벌 F&B 트렌드
/trend 밈                       → 밈/유행어 트렌드
/trend 패션                     → 패션 트렌드
/trend sns                     → SNS 6개 플랫폼 종합 트렌드
/trend 스레드 AI                → 스레드에서 특정 키워드 검색
/trend 인스타 #해시태그          → 인스타그램 해시태그 트렌드
/trend [URL]                   → 특정 URL 크롤링
```

---

## 실행 로직

### Step 1: 입력 파싱

`$ARGUMENTS`를 분석하여 모드를 결정합니다:

| 입력 | 모드 | 동작 |
|------|------|------|
| (없음) | **실시간 모드** | X 트렌딩 + SNS 6개 종합 |
| URL 포함 | **URL 모드** | 해당 URL 크롤링 |
| "sns" | **SNS 종합 모드** | 6개 플랫폼 트렌드 AI 분석 |
| "스레드" + 키워드 | **스레드 모드** | 스레드 키워드 검색 |
| "인스타" + 해시태그 | **인스타 모드** | 인스타 해시태그 트렌드 |
| "글로벌" 포함 | **글로벌 모드** | 글로벌 트렌드 서칭 |
| 키워드만 | **주제 모드** | X 트렌딩 + 웹 서칭 + (선택)SNS |

### Step 2: 분야별 최적 플랫폼 선택

캐릿 2026 트렌드 흐름 지도 기반 (Z세대 205명 설문):

| 분야 | 트렌드 시작 플랫폼 | 서칭 전략 |
|------|-------------------|----------|
| **밈/유행어** | 엑스(68%) > 인스타(61%) | X 트렌딩 + SNS 종합 |
| **핫플** | 인스타(91%) > 유튜브(31%) | 인스타 해시태그 + 웹 서칭 |
| **F&B** | 틱톡/샤오홍슈 → 인스타(85%) | SNS 종합 (틱톡+인스타) + 웹 |
| **패션** | 인스타(82%) > 핀터레스트(30%) | 인스타 해시태그 + SNS 종합 |
| **뷰티** | 인스타(69%) > 유튜브(60%) > 엑스(42%) | SNS 종합 + 웹 서칭 |
| **범용/기술/비즈니스** | 엑스 → 인스타 → 뉴스 | X 트렌딩 + 스레드 + 웹 서칭 |

## Step 3: 데이터 수집 도구 (Apify Actors)

### Actor 1: X 트렌딩 (빠른 실시간)
```
도구: mcp__apify-actors__call-actor
actor: "data-slayer/twitter-trends-by-location"
input: {"country": "SouthKorea"}
```
- 국가 옵션: "SouthKorea", "UnitedStates", "Japan", "UnitedKingdom" 등
- 결과: 실시간 트렌딩 토픽 50개 (이름, 카테고리)
- 비용: 매우 저렴

### Actor 2: SNS 6개 종합 트렌드 + AI 분석 (심층)
```
도구: mcp__apify-actors__call-actor
actor: "manju4k/social-media-trend-scraper-6-in-1-ai-analysis"
input: {
  "platforms": ["tiktok", "instagram", "youtube", "twitter", "reddit"],
  "region": "global",
  "timeRange": "24h",
  "maxTrends": 15,
  "includeMetrics": true,
  "enableComparison": true,
  "comparisonType": "comprehensive"
}
```
- 플랫폼: tiktok, instagram, youtube, twitter, reddit, pinterest (선택)
- 지역: "global", "US", "JP", "UK", "IN", "CA", "AU", "BR", "DE", "FR"
- 시간: "1h", "4h", "8h", "12h", "24h", "7d"
- 비용: $0.75/회 — **필요할 때만 사용** (실시간 모드 또는 "sns" 키워드일 때)
- 결과: 트렌딩 해시태그, 바이럴 콘텐츠, 인게이지먼트 비교, AI 추천

### Actor 3: 스레드 키워드/해시태그 검색
```
도구: mcp__apify-actors__call-actor
actor: "futurizerush/meta-threads-scraper"
input: {
  "mode": "search",
  "keywords": ["검색할 키워드"],
  "search_filter": "top",
  "max_posts": 20
}
```
- mode: "search" (키워드 검색), "keyword" (해시태그), "user" (유저 포스트)
- search_filter: "top" (인기순), "recent" (최신순)
- 날짜 필터: start_date, end_date (Search 모드에서만)
- 비용: 저렴 ($0.009/결과)

### Actor 4: 인스타그램 해시태그 트렌드
```
도구: mcp__apify-actors__call-actor
actor: "futurizerush/instagram-hashtag-scraper"
input: {
  "hashtags": ["운세", "타로"],
  "postsNewerThan": "7 days"
}
```
- 해시태그당 최대 24개 트렌딩 포스트/릴스
- 미디어 다운로드, 인게이지먼트 메트릭, 캡션 포함
- 비용: $0.005/결과 + $0.1/해시태그

### 도구 5: 웹 서칭
```
도구: WebSearch
query: "$ARGUMENTS 트렌드 2026"
```

### 도구 6: URL 크롤링
```
도구: WebFetch
url: "해당 URL"
prompt: "전체 내용을 추출해줘"
```

## Step 3 실행 전략 (모드별)

### 실시간 모드 (인수 없음) — 병렬 실행
1. **X 트렌딩** (Actor 1) — 한국 실시간 50개
2. **SNS 종합** (Actor 2) — 6개 플랫폼 24h 글로벌
3. **WebSearch** — "오늘 화제 트렌드"

### SNS 종합 모드 ("sns") — 병렬 실행
1. **SNS 종합** (Actor 2) — 전체 플랫폼
2. **X 트렌딩** (Actor 1) — 한국

### 스레드 모드 ("스레드" + 키워드)
1. **스레드 검색** (Actor 3) — 키워드 top 20

### 인스타 모드 ("인스타" + 해시태그)
1. **인스타 해시태그** (Actor 4) — 해시태그 트렌드

### 주제 모드 (키워드만) — 병렬 실행
1. **X 트렌딩** (Actor 1) — 한국 (키워드 매칭 확인)
2. **WebSearch** — "$ARGUMENTS 트렌드 2026"
3. **(선택)** 스레드 검색 (Actor 3) — 주제 키워드

### 글로벌 모드 ("글로벌") — 병렬 실행
1. **X 트렌딩** (Actor 1) — country: "UnitedStates"
2. **SNS 종합** (Actor 2) — region: "global"
3. **WebSearch** — "$ARGUMENTS global trend 2026"

### URL 모드
1. **WebFetch** — URL 크롤링

## 트렌드 흐름 참고 (캐릿 2026)

### 핵심 인사이트
- **플랫폼별 트렌드 분화**: "인스타는 핫플, 엑스는 밈" — 분야별 분리
- **Z세대 91.7%**: "나이보다 라이프스타일이 같으면 같은 트렌드를 공유"
- **트렌드 흐름**: 엑스/틱톡(시작) → 인스타/유튜브(대중화) → 블로그/뉴스(정착)
- **마이크로 트렌드 시대**: 여러 마이크로 트렌드가 공존 (75.6%)
- **오프라인 트렌드 부상**: 러닝크루, 소셜링 등 (74.1%)
- **C트렌드**: 샤오홍슈 → 엑스/틱톡 → 국내 확산 (40% 경험)

### 플랫폼별 특성
- **엑스**: 덕후 문화, 인용RT 밈 확산, "찐템" 인식, 트렌드 시작점
- **인스타그램**: 대중 SNS, 핫플/패션/뷰티 확산, DM 공유 릴스 바이럴
- **스레드**: 텍스트 중심, 초기 도달률 높음, 밈/의견 공유에 강점
- **틱톡**: 글로벌 트렌드 창구, F&B/챌린지 발상지
- **핀터레스트**: 패션 원류 (해외 룩북)
- **샤오홍슈**: 뷰티/C트렌드 핵심
- **유튜브**: 쇼츠는 하류, 롱폼은 심층 리뷰
- **레딧**: 니치 커뮤니티, 기술/게임/문화 트렌드

### 트렌드 리딩 그룹 (분야별)
- **밈**: 엑스 헤비유저, 아이돌/애니 덕후
- **핫플**: 인스타 인플루언서, 매거진 계정
- **F&B**: 먹플루언서 (젼언니, 프롬서희 등), 글로벌 틱톡커
- **패션**: 인스타 패션 인플루언서, 핀터레스트 크리에이터
- **뷰티**: 코덕, 유튜브 뷰티 크리에이터, 샤오홍슈 유저

## 비용 관리

| Actor | 비용 | 사용 시점 |
|-------|------|----------|
| X 트렌딩 | ~$0.01 | 항상 (저렴) |
| SNS 6개 종합 | **$0.75/회** | "sns" 키워드 또는 실시간 모드 |
| 스레드 검색 | ~$0.10+$0.009×N | "스레드" 키워드일 때 |
| 인스타 해시태그 | ~$0.10+$0.005×N | "인스타" 키워드일 때 |
| WebSearch | 무료 | 항상 |
| WebFetch | 무료 | URL 모드 |

**비용 절감 팁**: 주제 모드에서는 X 트렌딩 + WebSearch만 사용 (무료~$0.01). SNS 종합($0.75)은 명시적 요청 시에만.

## 주의사항

- Apify Actor 호출 시 `async: false`로 즉시 결과 받기
- SNS 종합 Actor는 비용이 있으므로 필요할 때만 사용
- 스레드/인스타 Actor는 성공률이 100%가 아닐 수 있음 — 실패 시 WebSearch로 대체
- 결과가 너무 많으면 상위 20~30개로 제한
- 병렬 실행 가능한 Actor는 동시에 호출하여 속도 최적화
